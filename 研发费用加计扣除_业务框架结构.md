## 模块一：基础配置管理（一期）

> 全局参数配置，为辅助账编制提供规则引擎和基础数据体系。

### 1.1 标准科目映射
- 企业会计科目 ↔ 研发费用标准科目映射
- 三口径管理：会计核算、加计扣除、高新认证
- 标准科目增加管理用途标记
- 科目设置原则：不同口径归集范围存在差异的，必须单列明细科目

### 1.2 费用分摊规则
- **分摊对象**：人员人工费用、资产折旧、无形资产摊销、材料费用、其他摊销费用
- **分摊动因字典**：人员工时、机器工时、场地面积、营业收入、经营收益、研发投入、协议比例、其他、不适用
- **动因数据源**：人员工时记录表、设备运行记录表、预设分摊比例表
- **规则内容**：规则编码、规则名称、分摊动因、动因数据源、分摊比例公式、标准科目范围、费用明细数据源、有效期
- **约束**：分摊方法在一个纳税年度内不可随意变更，需留存审批记录

### 1.3 限额计算规则
- **限额场景**：加计扣除（10%）、高新认证（20%）等
- **规则要素**：限额费用取数科目、限额比例、限额基数取数科目、有效期、政策依据文号
- **适用粒度**：按项目、按总额（公司组织）
- **通用性**：各地区、各公司、各项目适用一套

### 1.4 辅助账样式
- 一期先做加计扣除2021版（财税〔2021〕A类107012表样式）

### 1.5 辅助账差异阈值
- 按公司配置
- 用于费用明细数据源与研发费用明细账比对时的异常判断条件

## 模块二：项目信息管理（一期）

> 管理研发项目全生命周期及关联的人员、资产、组织等主数据。

### 2.1 研发项目信息
- **项目状态流转**：已立项 → 研发中 → 已结项
- **支出类型**：费用化、资本化、混合型
- **组织形式**：自主研发、委托研发、合作研发、集中研发
- **资本化结转操作**：已结项 → 已关账 → 资本化阶段"归集核算" → 完成结转 → 生成无形资产
- **关键字段**：项目编号、名称、负责人、承担部门、技术领域、总预算、实际支出、立项/开始/结束日期、资本化阶段、完成情况、委托关系、管理用途等
- **管理用途**：加计扣除、高新认定、IPO上市

### 2.2 研发活动看板（二期）
- 研发项目数量统计（新立项、研发中、已结项）
- 不同口径研发费用金额
- 限额费用超比例情况

### 2.3 主数据管理
| 功能 | 说明 |
|------|------|
| 研发人员信息 | 项目参与人员名单 |
| 研发资产信息 | 研发使用的仪器设备等 |
| 研发组织信息 | 用于集中研发费用分摊，可在项目中直接维护 |
| 研发档案信息 | 项目备查资料管理 |
| 研发成果信息 | 二期规划 |
| 研发辅助判断 | 二期规划 |

## 模块三：辅助账管理（一期核心）

> 从数据采集到辅助账生成的全流程业务处理，是系统最核心的模块。

### 3.1 账簿管理
- **核心实体**：公司 → 研发项目 → 辅助账（一一或一对二）
- **账簿状态**：待建账 → 已启用 → 已关账
- **配置项**：启用月份、结账周期（月/季/年）、记账方式（汇总/明细）
- **自动建账**：有项目信息但无辅助账信息时，自动产生"待建账"记录
- **混合型项目**：生成二条待建账记录（费用化 + 资本化）

### 3.2 数据源管理
#### 3.2.1 分摊动因数据源
- 每期分摊对象数据维护
- 工资数据维护由单独权限管理
- 一期可先实现部分动因场景

#### 3.2.2 费用明细数据源（10类）
| 序号 | 数据源类型 | 用途 |
|------|-----------|------|
| 1 | 科目余额表 | 数据输入 |
| 2 | 科目辅助余额表 | 规则引用 |
| 3 | 科目明细账 | 计算分摊结果 |
| 4 | 科目辅助明细账 | 生成分摊分录 |
| 5 | 工资薪酬明细表、工时分配表 | 分配计算 |
| 6 | 固定资产折旧明细、设备使用记录表 | 折旧分摊 |
| 7 | 无形资产折旧明细、分配表 | 摊销计算 |
| 8 | 仪器设备租赁费计提表、费用报销单 | 租赁及报销 |
| 9 | 费用分割单 | 分摊依据 |
| 10 | 物料领用台账 | 材料投入 |

- **采集方式**：接口对接、数据导入、手工录入

### 3.3 研发费用明细账
- 基于企业研发费用明细账表ODS
- 按"企业会计科目 + 辅助核算项目"汇总凭证数据

### 3.4 辅助账生成流程（核心流程）

```
数据初始化 → 数据校验 → 自动/手工分摊 → 费用归集 → 生成辅助账 → 审核
```

**Step 1：数据初始化**
- 基于 ODS 明细账数据源，汇总借方/贷方金额

**Step 2：数据校验（源数据校验）**
- 明细账数据与费用台账比对
- 异常判断取自"辅助账差异阈值"配置
- 除研发费用明细账外，其他数据源均需匹配和总额比对

**Step 3：费用归集计算**
- 按标准科目映射规则，将企业科目数据归集到研发标准科目

**Step 4：费用分摊计算**
- **自动分摊**：分摊比例表为空的明细行，系统按分摊规则自动计算
- **手工分摊**：手工选择分摊比例表进行分摊
- **取消分摊**：组织编号、会计期间、科目、辅助核算项目、分摊比例表均相同时可操作

**Step 5：生成辅助账**
- 按项目、科目维度生成辅助账明细行
- 辅助账凭证编号汇总被汇总的全部凭证（逗号分隔）

**Step 6：手工调整**
- 调整分录通过"调整辅助账"新增，凭证种类标记为"辅助专用"
- 需上传附件，按档案类别分类

**Step 7：审核**
- 状态流转：待分摊 → 待审核 → 已审核 / 已作废

### 3.5 关联台账
| 台账 | 说明 |
|------|------|
| 工资薪酬明细表 | 人员人工费用分摊依据 |
| 折旧摊销明细表 | 资产折旧/无形资产摊销分摊依据 |
| 物料领用台账 | 材料直接投入 |
| 费用报销台账 | 其他费用归集 |

### 3.6 资本化项目摊销计算
- 资本化结转部分的无形资产摊销自动计算

### 3.7 其他费用限额计算
- **计算层级**：公司口径 → 项目口径（资本化项目二次分摊）
- **操作流程**：新建 → 刷新计算 → 确认
- **限额金额** = 限额基数金额 × 限额比例
- **资本化项目调整后费用** = 公司调整后其他相关费用 / 公司账载金额 × 资本化项目账载金额
- 已确认数据不可刷新

### 3.8 查询与报表
| 功能 | 一期/二期 | 说明 |
|------|----------|------|
| 辅助账查询 | 一期（2021版） | 按项目、期间查询，按不同样式/口径（二期） |
| 辅助账汇总表查询 | 一期 | 按公司、期间查询 |
| 加计扣除明细表查询 | 一期 | 按期创建、保存、查询（107011表） |
| 辅助账导入 | 二期 | 支持外部数据导入辅助账 |

### 3.9 对账管理（二期）
- 科目余额表与辅助账比对
- 辅助账与总账校验

### 3.10 结账管理（二期）
- 期末结账、关账、反结账流程

## 业务数据流转全景

```
┌──────────────────────────────────────────────────────────────┐
│                    【基础配置层】                               │
│  标准科目映射 ←→ 费用分摊规则 ←→ 限额计算规则 ←→ 辅助账样式     │
└───────────────────────────┬──────────────────────────────────┘
                            │ 规则驱动
┌───────────────────────────▼──────────────────────────────────┐
│                    【数据采集层】                               │
│  企业研发费用明细账ODS │ 工资薪酬表 │ 折旧明细 │ 物料台账 │ 费用报销│
└───────────────────────────┬──────────────────────────────────┘
                            │ 接口/导入/手工
┌───────────────────────────▼──────────────────────────────────┐
│                    【数据校验层】                               │
│  明细账 ↔ 费用台账 比对校验 → 异常提示（差异阈值）              │
└───────────────────────────┬──────────────────────────────────┘
                            │
┌───────────────────────────▼──────────────────────────────────┐
│                    【计算处理层】                               │
│  费用归集计算 → 费用分摊计算 → 其他费用限额计算                  │
│                   ↑ 分摊动因数据                                │
│  人员工时/机器工时/场地面积/营业收入/研发投入/协议比例...         │
└───────────────────────────┬──────────────────────────────────┘
                            │
┌───────────────────────────▼──────────────────────────────────┐
│                    【辅助账生成层】                              │
│  自动生成辅助账 → 手工调整 → 审核 → 对账（二期）→ 结账（二期）   │
└───────────────────────────┬──────────────────────────────────┘
                            │
┌───────────────────────────▼──────────────────────────────────┐
│                    【税务申报输出层】                            │
│  2021版辅助账明细表(107012)                                    │
│  2021版辅助账汇总表                                            │
│  研发费用加计扣除优惠明细表(107011)                             │
└──────────────────────────────────────────────────────────────┘

项目信息管理（项目全生命周期）贯穿以上所有层级
账簿管理（建账/启停/关账）控制辅助账管理的运行边界
```

## 分期规划总结

| 阶段 | 模块 | 优先级 |
|------|------|--------|
| **一期** | 基础配置管理（科目映射、分摊规则、限额规则、辅助账样式、差异阈值） | 高 |
| **一期** | 项目信息管理（项目信息、人员/资产/组织/档案） | 高 |
| **一期** | 辅助账管理（账簿管理、数据源、校验、归集、分摊、生成、查询） | 高 |
| **二期** | 研发活动看板、研发成果、研发辅助判断 | 中 |
| **二期** | 对账管理、结账管理、辅助账导入 | 中 |
| **二期** | 后续管理（事后核查） | 中 |
| **二期** | 管理报表、风险管理、政策案例 | 中 |
