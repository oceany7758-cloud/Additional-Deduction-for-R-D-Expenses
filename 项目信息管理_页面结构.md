### 1.3 资本化结转操作页

**前置条件检查：**

| 条件 | 值 |
|------|-----|
| 项目状态 | 已结项 |
| 辅助账状态 | 已关账 |
| 资本化阶段 | 归集核算 |

**结转流程（8步）：**

1. 获取费用化阶段辅助账数据
2. 获取资本化阶段辅助账数据
3. 计算资本化金额
4. 生成结转凭证
5. 更新资本化阶段为"完成结转"
6. 生成无形资产卡片
7. 关联无形资产摊销计算
8. 更新辅助账状态

---

### 1.4 资本化阶段自动赋值逻辑（业务规则18）

| 支出类型 | 资本化阶段取值逻辑 |
|----------|-------------------|
| 费用化 | 始终为"不适用" |
| 资本化 | 【资本化开始日】前 → "未开始"；【资本化开始日】至【资本化结转】操作完成前 → "归集核算"；【资本化结转】操作完成后 → "完成结转" |
| 混合型 | 同资本化（需先填写【资本化开始日】，该字段为必填） |

## 数据关联关系

```
研发项目信息（主表，详情30字段）
  ├─ 1:N → 项目参与人员 ← 关联 → 研发人员信息（15字段）
  │        展示：工号、姓名、专业、职称名称、职务、用工类别、研发分工、参与项目起止
  │        约束：工时已计入项目的人员不可解除关联
  ├─ 1:N → 项目使用资产 ← 关联 → 研发资产信息（19字段）
  │        展示：资产编号、资产名称、资产类别、用途分类
  │        约束：折旧/摊销费已计入项目的资产不可解除关联
  ├─ 1:N → 项目参与组织 ← 关联 → 研发组织信息（8字段）
  │        展示：组织名称、证件号码
  │        联动：选择组织后，委托研发的关联关系、是否委托境外自动赋值
  ├─ 1:N → 项目备查资料 ← 自动归集 → 研发档案信息（19字段）
  │        展示：项目阶段、源文件编号、档案名称、档案类别、档案日期、附件
  └─ 1:1 → 账簿管理（辅助账状态同步）

研发资产信息（19字段）
  └─ N:M → 研发项目信息（所涉研发项目，二期）

研发档案信息（19字段）
  └─ N:M → 研发项目信息（项目编号，允许多选）
```

## 系统自动赋值字段汇总

| 字段 | 赋值逻辑 | 来源 |
|------|---------|------|
| 资本化阶段 | 根据支出类型+资本化开始日+结转操作状态自动赋值 | 见1.4节逻辑 |
| 辅助账状态 | 同步账簿管理中对应账簿的状态 | 账簿管理 |
| 项目实际支出 | = 费用化支出金额 + 资本化支出金额 | 研发支出辅助账 |
| 职称类别 | 根据职称名称自动匹配（38项映射） | 2.2节映射表 |
| 委托研发的关联关系 | 从组织信息关联选择时自动赋值 | 研发组织信息 |
| 是否委托境外 | 从组织信息关联选择时自动赋值 | 研发组织信息 |
| 档案用途（默认值） | 根据档案类别小类的关联关系自动填充 | 5.2节映射表 |
